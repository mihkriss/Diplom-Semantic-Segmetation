# Diplom: Semantic Segmentation & Generative Models

> **ВКРМ** по теме "Разработка системы нейросетевой генерации изображений городских сцен на основе семантических карт"
> (Stable Diffusion + ControlNet + LoRA)

## 📌 О проекте

Данный репозиторий содержит исследования и разработки в области компьютерного зрения. Основные направления:

- Обучение и оценка моделей **семантической сегментации**
- Генерация синтетических размеченных данных с помощью **Stable Diffusion**, **ControlNet** и **LoRA**
- Сравнение метрик качества сегментации на реальных и синтезированных изображениях
- Создание прототипа интерфейса для генерации изображений с масками

Проект выполнен в формате Jupyter Notebook для воспроизводимости экспериментов.

## Основные возможности
- Обучение/тонкая настройка модели сегментации на кастомном датасете
- Использование **ControlNet** для управления генерацией изображений по картам сегментации
- Применение **LoRA** для эффективной дообучения генеративных моделей
- Визуализация метрик и сравнение качества сегментации
- Простой интерфейс для генерации синтетических пар (изображение + маска)

## Используемые технологии

- Python 3.11
- PyTorch 
- Hugging Face Diffusers (для Stable Diffusion)
- ControlNet
- PEFT (LoRA)
- OpenCV, Matplotlib, NumPy
